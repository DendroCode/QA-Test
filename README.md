# README for QA-trainee-assignment-autumn-2025

---

Привет! Здесь вы сможете узнать о структуре проекта, а также запустить тесты и проверить работу микросервиса. 

## О проекте
Это тестовое задание для QA-стажёра. Внутри можно найти автотесты для микросервиса объявлений Avito, а также багрепорты по ошибкам со скриншота сайта вакансий Avito. 

## Структура
- `test_1.py` - тест создания объявления
- `test_2.py` - тест получения объявления по ID
- `test_3.py` - тест получения всех объявлений продавца
- `test_4.py` - тест получения статистики
- `test_5.py` - тест проверки существующих объявлений на ошибки
- `test_6.py` - тест создания объявления с нулевой статистикой
- `quest_1.md` - найденные баги сайта вакансий Avito
- `TESTCASES.md` - тест-кейсы
- `BUGS.MD` - найденные баги в работе микросервиса
- `requirements.txt` - зависимости
- `README.md` - описание проекта (вы здесь)

## Инструкция по запуску автотестов микросервиса
1. Склонируйте к себе репозиторий тестового задания через выполнение команды в терминале:
```
  git clone https://github.com/DendroCode/QA-Test.git
```
Или скачайте ZIP-архив по [ссылке](https://github.com/DendroCode/QA-Test/archive/refs/heads/main.zip). 

2. Проверьте, установлен ли у вас Python. В командной строке/терминале выполните команду:
```
  python --version
```  
Если он не установлен, то установите с официального [сайта Python](https://www.python.org/downloads/), выбрав подходящую версию для вашей операционной системы, и пройдите шаг сначала.  В процессе установки обязательно поставьте галочку в чекбоксе «Add python.exe to PATH».

3. Через командную строку/терминал перейдите в корневую директорию проекта, выполнив команду:
```
  cd /здесь укажите путь до директории с проектом
```

4. Установите необходимые зависимости из файла `requirements.txt`, выполнив команду  
```
  pip install -r requirements.txt
```
если она не выполняется, то попробуйте
```
  pip3 install -r requirements.txt
``` 
5. Запустите тесты, выполнив команду:
```
  pytest -v
```
